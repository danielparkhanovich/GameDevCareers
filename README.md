### Features include

- mails (via SMTP)

### Technologies
- ASP.NET Core MVC
- MS SQL Server with EF Core (ORM)
- Azure (App Service, Blob Storage)
- Redis
- xUnit
- FluentValidation
- HTML, CSS, JavaScript
- JQuery
- Bootstrap
